Community Guidelines

Welcome

Acetyl Client is meant to be a fun, energetic, and welcoming chat app for everyone. Whether you’re here to chat with friends, join new communities, or explore what this platform can do, these guidelines help keep things safe, respectful, and enjoyable. Think of this page as both a quick beginners guide and an agreement on how we treat each other.

Getting Started

You can join the Public server instantly, or enter a server code to join private spaces with friends or groups. Creating your own server is easy too — just pick a name and share the code with whoever you want to invite. If you ever forget the join code for a server, just hover over the server in the sidebar to see it.

Messages support normal text, special characters, emojis, files, and anything else you can think of. Feel free to express yourself and bring some personality into the chat. Just keep it readable and avoid flooding the conversation (aka spamming). If you want to send a direct message to someone, you can do so by including @[their username] at the start of the message.

If something isn’t working, you have ideas, or you want to request a feature, you can always use the feedback form. It’s there for everything: bug reports, suggestions, improvements, or general thoughts about the platform.

The Actual Guidelines

Be respectful. Treat everyone like a real person. No harassment, hate speech, or anything like that.

Keep things clean and safe. No illegal content, harmful links, malware, or anything that could put others at risk.

Have fun, but don’t disrupt. You are welcome to express yourself, but spamming is not allowed. It’s annoying and disruptive for everyone. Use common sense. If you’re unsure whether something is okay, it’s probably better not to send it.

Safety & Moderation

Admins help keep the platform safe and enjoyable. They can step in when needed, remove harmful content, and issue bans. If you’re banned, you may be able to appeal depending on the situation.

If you run into a problem — whether it’s a bug, a user issue, or something that feels off — you can always use the feedback form to report it. It’s the best way to get help, request fixes, or suggest improvements.
